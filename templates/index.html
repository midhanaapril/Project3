List of API's

For All Breweries
/beer_api

This API includes all beer data

For All Breweries
/beer_api/member/< membership >

This API allows you to filter breweries by membership.

Available memberships: Micro, Taproom, Brewpub, Large, Contract, Regional, Proprietor

For All Breweries
/beer_api/member/< brewery name >

This API returns a single brewery detail based on name.

For A List of Brewery Names
/beer_api/brewery_names

This API returns a list of brewery names.
